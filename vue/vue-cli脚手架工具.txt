vue-cli 脚手架的工具
    自动帮我们创建vue应用目录

    下载安装
        全局
        npm install vue-cli -g

    使用
        vue init 模板名 应用目录名，文件名

        vue中使用模板
            simple          最简单的模式(没用！)
            webpack         最大的项目
            webpack-simple  较小的项目

        // 初始化创建一个使用simple模板的vue应用
        vue init simple vue-simple-demo

        vue init webpack vue-webpack-demo
            1. 创建应用目录
            2. 进入目录，下载依赖包

            注意：ESLint 检测JS语法结构
                Use ESLint to lint your code? (Y/n) n

	运行
		npm run dev

    版本
        $ vue --version
        2.8.2


npm init        新建一个.json文件
npm list        查看所有安装的包
npm prune        移除项目不依赖的包
npm outdated        查看哪些包过时了
