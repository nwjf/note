

## 项目结构-建议
> 只为大型模块化系统代码目录建议

```shell
|-- bin
|   |-- www             # 入口文件
|-- public              # 公共资源
|-- routes              # 路由目录
|-- views               # 视图目录
|
|-- controller          # 控制器
|-- service             # 业务层
|-- model               # 对象层
|-- middelware          # 中间层-中间件
|-- config              # 配置文件
|-- app.js              # 跟入口
|-- package.json        # 
```
